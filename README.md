## Customization

### Colors
Edit the `:root` section in `styles.css`:

```css
:root {
    --primary: #1a1a1a;      /* Dark grey text */
    --accent: #0066cc;        /* Blue links/buttons */
    --light: #f8f8f8;         /* Light backgrounds */
}
```

### Add Your Photo
Find this in `index.html`:

```html
<!-- Add after "About Me" section -->
<img src="photo.jpg" alt="Shahnaz" style="width: 200px; border-radius: 8px;">
```

### Edit Content
All content is in `index.html` between `<h2>` and `</h2>` tags.
